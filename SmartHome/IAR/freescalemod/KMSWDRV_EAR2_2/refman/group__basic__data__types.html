Detailed Description

The Kinetis-M Peripheral Drivers defines several basic data types to support code portability between different hardware architectures and tools. These basic data types, which are defined in the C header file common.h supports generic word and integer data types. This is used throughout the interface definitions for the peripheral drivers.
