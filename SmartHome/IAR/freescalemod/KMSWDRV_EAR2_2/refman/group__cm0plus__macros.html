#define UserMode ( )

This macro enters unprivileged (user) mode of execution. It can be called from software running in privileged (supervisor) mode.

Note
Implemented as an inline macro.
See Also
SupervisorMode, SVC_Call, ProcessStackPointer, MainStackPointer
#define SupervisorMode ( )

This macro enters privileged (supervisor) mode of execution. It can be called from software running in handler or privileged (supervisor) modes.

Note
Implemented as an inline macro.
Warning
Calling it from unprivileged (user) mode of execution causes not entering into privileged (supervisor) mode of execution. Call SVC_Call() from privileged to user mode.
See Also
UserMode, SVC_Call, ProcessStackPointer, MainStackPointer
