Detailed Description

This section specifies default configuration structures for PORT module. Create a new configuration structure if default ones don't initialize on-chip peripheral in required operating mode (see Creating Application Specific Peripheral Configuration Structure).

Macro Definition Documentation

#define PORT_MODULE_ALT0_MODE
Value:
(tPORT){ \
/* PCR */ SET(PORT_PCR_IRQC(0))|CLR(PORT_PCR_LK_MASK)|SET(PORT_PCR_MUX(0))| \
/* ... */ CLR(PORT_PCR_SRE_MASK)|CLR(PORT_PCR_PE_MASK)| \
/* ... */ CLR(PORT_PCR_PS_MASK), \
/* DFER */ CLR(PORT_DFER_DFE_MASK), /* use mask for all pins */ \
}

Configures port pin(s) with the following characteristics:

  • Internal pull-up disabled
  • Digital filter disabled
  • Interrupt disabled
  • Slew rate control disabled

The initialization value of the configuration structure split by peripheral registers is shown above.
