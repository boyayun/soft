Detailed Description

This section specifies the exact usage for each I/O API macro. All logical operations, which destination peripheral register address reg is in peripheral address space [0x4000_0000-0x4007_FFFF], are accelerated by the Bit Manipulation Engine (BME), emulated by software otherwise. The core platform's Bit Manipulation Engine (BME) provides hardware support for atomic read-modify-write memory operations to the peripheral address space.

Warning
The basic logical operations are either accelerated by the Bit Manipulation Engine (BME), if address of the destination on-chip peripheral register reg is in range from 0x40000000 to 0x4007FFFF, or they're emulated by software.
