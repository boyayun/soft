SYST Software Execution Measurement
SYST Examples

This example demonstrates use of the System Timer (SYST) module for software execution measurement.

Source code:
/******************************************************************************
* (c) Copyright 2010-2013, Freescale Semiconductor Inc.
* ALL RIGHTS RESERVED.
******************************************************************************
* systmsr_test.c
******************************************************************************/
#include "drivers.h"
static uint32 volatile ticks;
void main (void)
{
/* initialize system Timer */
SYST_Init (SYST_MODULE_FREERUN_TIMER_EN_CONFIG,PRI_LVL0,NULL);
/* clear system Timer */
SYST_ClrTimer ();
__asm volatile ("nop");
__asm volatile ("nop");
__asm volatile ("nop");
/* read value from the system Timer */
ticks = SYST_GetTimer (); /* ticks = 3 core clock cycles */
while(1);
}
