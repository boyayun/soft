CRC Example
CRC Driver

This example demonstrates typical use of the Cyclic Redundancy Check (CRC) module to calculate commonly used and standardized CRCs such CRC-32, CRC-16-CCIT, CRC-16-DNP and CRC-16-ANSI.

Source code:
/******************************************************************************
* (c) Copyright 2010-2013, Freescale Semiconductor Inc.
* ALL RIGHTS RESERVED.
******************************************************************************
* crc_test.c
******************************************************************************/
#include "drivers.h"
static const uint8 crcdata[]={'h','e','l','l','o',' ','w','o','r','l','d','!'};
static vuint32 result;
void main (void)
{
SIM_Init (SIM_MODULE_ALL_PERIPH_ON_CONFIG);
CRC_Init (CRC_MODULE_CRC32_CONFIG);
result = CRC_Calc (crcdata, sizeof(crcdata));
CRC_Init (CRC_MODULE_CRC16_CCIT_CONFIG);
result = CRC_Calc (crcdata, sizeof(crcdata));
CRC_Init (CRC_MODULE_CRC16_DNP_CONFIG);
result = CRC_Calc (crcdata, sizeof(crcdata));
CRC_Init (CRC_MODULE_CRC16_ANSI_CONFIG);
result = CRC_Calc (crcdata, sizeof(crcdata));
while(1);
}
