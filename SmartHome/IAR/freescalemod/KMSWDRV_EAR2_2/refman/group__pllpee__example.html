PLL Engaged Controlled by 32.768 kHz RTCOSC
PLL Examples

This example demonstrates PLL Engaged External (PEE) mode clocked by 32.768 kHz RTC Oscillator.

Source code:
/******************************************************************************
* (c) Copyright 2010-2013, Freescale Semiconductor Inc.
* ALL RIGHTS RESERVED.
******************************************************************************
* pllpee_test.c
******************************************************************************/
#include "drivers.h"
void main (void)
{
/* enable clocks to all on chip peripherals */
SIM_Init (SIM_MODULE_ALL_PERIPH_ON_CONFIG);
/* route core clock to PTF7 for monitoring */
SIM_SelClkoutSrc (CLKOUT_SRC1);
PORT_Init (PORTF,PORT_MODULE_ALT3_MODE,PIN7);
/* PLL settings */
PLL_Init(PLL_MODULE_PEE_12_288MHZ_CONFIG);
while(1);
}
