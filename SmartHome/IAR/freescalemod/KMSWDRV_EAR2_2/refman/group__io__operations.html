Detailed Description

This section describes logical operations making direct access to I/O registers performed by core platform's Bit Manipulation Engine (BME). By default, software drivers use BME read-modify-write capability for on-chip peripheral programming. The BME support can be disabled by inserting #define EMULATE_BME_IO_ACCESS into appconfig.h file.
