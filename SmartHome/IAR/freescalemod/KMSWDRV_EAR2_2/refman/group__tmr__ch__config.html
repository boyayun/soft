Detailed Description

This section specifies default configuration structures for TMR module. Create a new configuration structure if default ones don't initialize on-chip peripheral in required operating mode (see Creating Application Specific Peripheral Configuration Structure).

Macro Definition Documentation

#define TMR_CH_CNTR_CONFIG1 (   prm_clk_src,
  sec_clk_src 
)

By default Output mode is set to "Asserted while counter is active" By default Input Clock source selected is "Count Rising Edge of Primary Source" This use case will configure the QTimer in Counter Mode with following characteristics:

  • Counts up
  • Rollover enabled
  • One shot Timer mode disabled
  • Overflow interrupt disabled
  • Co-channel initialization disabled
  • Compare interrupt enabled
  • Input capture interrupt disabled
  • Input signal polarity not inverted
    Parameters
    prm_clk_srcSelect one of the following TMR Primary Count Sources
    sec_clk_srcSelect one of the following TMR Secondary Count Sources

#define TMR_CH_OUTCMP_CONFIG (   count_mode,
  prm_clk_src,
  sec_clk_src,
  output_mode 
)

This use case will configure the QTimer in Output Compare Mode with following characteristics:

  • Counts up
  • Count until compare then re-initialize.
  • One shot Timer mode disabled
  • Overflow interrupt disabled
  • Co-channel initialization disabled
  • Compare interrupt enabled
  • Input capture interrupt disabled
  • Input signal polarity not inverted
  • Master operation disabled
  • External OFLAG force disabled.
  • Polarity of OFLAG output signal not inverted.
  • Output not forced to external pin
    Parameters
    count_modeSelect one of the following TMR Count Modes
    prm_clk_srcSelect one of the following TMR Primary Count Sources
    sec_clk_srcSelect one of the following TMR Secondary Count Sources
    output_modeSelect one of the following TMR OFLAG Signal Modes of Operation
